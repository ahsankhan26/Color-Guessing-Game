# How RGB Works
RGB stands for "Red", "Green" & "Blue".

The combination of these three colors can make any color in the color spectrum.

The values range from 0-255 for each channel.

The representation is (R,G,B)

# Examples

| Color | RGB           |
| ----- | ----          |
| Black | (0,0,0)       |
| White | (255,255,255) |
| Red   | (255,0,0)     |
| Green | (0,255,0)     |
| Blue  | (0,0,255)     |
